# yooncom(커뮤니티 사이트)
---------- 
## What is yooncom 
> React를 사용하여 제작한 다른사람들이 적은 글을 보거나 내글을 작성 할수있는 커뮤니티 플랫폼입니다.
----------
## 사용한 기술 
> + HTML
> + CSS
> + JavaScript
> + React
> + StyledComponent
> + Firebase
> + ReactRouter
---------
## 구현한 기능 
> + 구글로그인, 로그아웃
> + 글 추가.삭제 
> + 최근글보기
> + 내가쓴글보기
> + 트랜드글 보기
> + 북마크 글 보기
> + 글 좋아요 기능 
> + 글 북마크 기능 
> + 글 전체 내용 보기
----------
